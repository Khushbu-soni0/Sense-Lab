Welcome to Sense Lab

This website provides a result of Infection of any disease with blood cell images. For this we have gathered datasets which include segmented cells from thin blood smear slide images that were acquired from the Disease Screener research operation using a smartphone camera.

We have applied ensemble learning on the given dataset in this project because ensemble techniques combine the various models and provide a better prediction compared to the single model.
